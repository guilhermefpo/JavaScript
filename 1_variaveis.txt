 ° Um sinal de igual(=) significa RECEBE.

 °  vaga a1 = carro1
         a1 = carro2
         a1 = null("Valor nulo")

 ° Variáveis: Espaços na memória do computador em JavaScript, se atribui uma Variável com a palavra ("var"), também podemos usar a palavra ("let").
       Exemplo:  var n1 = 5
                 var n2 = 8.5
                 var n3 = 15

 . Pode ter Variáveis de tamanhos diferentes para caber dados de tamanhos diferentes.
        Exemplo: var s1 = "JavaScript"
                 var s2 = 'Curso em Vídeo'
                 var s3 = `Guanabara`

 . O nome de cada Variável se chama de "Identificador", existe regras para nomeação.
  - 1: Podem começar com LETRA, $, _
  - 2: Não podem começar com números.
  - 3: É possível usar letras ou números.
  - 4: É possível usar acentos e símbulos.
  - 5: Não pode ter espaços.
  - 6: Não podem ser palavras reservadas.

  DICAS PARA IDENTIFICADOR:
  - 1: Maiusculas e minusculas fazem diferença.
  - 2: Nomes coerentes de acordo com os problemas.

  * Variáveis serve para guardar dados de diferentes tipos.
  - number - Infinity     NaN
  - string
  - boolean
  - Object - Array    

= Para saber esses tipos usa-se comando ("typeof").
